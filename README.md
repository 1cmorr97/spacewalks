# Spacewalks

## Overview

Spacewalks is a python analysis tool for researchers to generate visualizations and statistacl summaries of Nasa's extravehicular activity datasets.

## Features
Key features of Spacewalks:

- Generates a csv table of summary statistics of EVA crew sizes
- Generates a line plot to show the cumulative duration of space walks over time

## Pre-requisites
Spacewalks was developed using python version 3.12
To install and run Spacewalks, you will need to have Python >= 3.12 installed.
You will also need the following libraries
- [Numpy](https://www.numpy.org) >=2.0.0 - Spacewalks' test suite uses numpy's statistical functions
